# Прокси для скрапера — по одному на строку
# Поддерживаемые форматы:
#   http://host:port
#   http://user:password@host:port
#   socks5://host:port
#   socks5://user:password@host:port
#
# Строки начинающиеся с # — комментарии, игнорируются
# Пустые строки тоже игнорируются

# Пример:
# http://123.45.67.89:8080
# http://login:pass@123.45.67.89:3128
# socks5://123.45.67.89:1080
